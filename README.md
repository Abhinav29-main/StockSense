# StockSense - AI-Powered Market Intelligence

## Track: Claw & Shield

## Features
- Real-time stock data via yfinance
- Random Forest ML predictions
- FinBERT sentiment analysis on news
- Technical indicators (RSI, MACD, SMA, Bollinger)
- Portfolio tracker
- OpenClaw agent interface
- ArmorIQ security (intent tokens, fail-closed policy)

## Setup

```bash
# Install dependencies
pip install -r requirements.txt

# Run the launcher
python run_stocksense.py
